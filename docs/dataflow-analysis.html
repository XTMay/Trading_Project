关闭所有可能干扰自动化抓取的 Excel 特性（链接更新弹窗、自动恢复、错误检查）， 清除上次残留的网络查询连接，确保干净状态。

3 Phase 1: 单股查询 — 主数据流

这是系统最核心、最常用的路径。以台股为完整示例（美/港/中/全球结构相同）。

10.2.3 P2: URL 配置外部化

问题： 所有 URL 硬编码在 Macro 函数体内，任何网站改版都需要修改代码。

方案： 将 URL 模板提取到配置 Sheet 或常量模块：
